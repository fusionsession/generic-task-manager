# Generic Task Manager

## Project Overview

**Generic Task Manager** is a lightweight web-based task management application developed using HTML, CSS, and Vanilla JavaScript. The application allows users to create, manage, organize, and track tasks through a simple and responsive interface.

The project demonstrates the implementation of CRUD operations, dynamic DOM manipulation, client-side data persistence, and responsive web design principles.

---

## Features

### Task Management (CRUD)

* Create new tasks
* Read and display task information
* Update existing tasks
* Delete tasks

### Additional Functionality

* Mark tasks as completed
* Display task creation date and time
* Move tasks up and down within the list
* Automatically count completed tasks
* Store data in LocalStorage
* Responsive user interface
* Text formatting support:

  * Bold
  * Italic
  * Underline

## Data Storage

The application uses the browser's LocalStorage to persist task data between sessions. No external database or backend server is required.

---

## How to Run

1. Download or clone the project repository.
2. Open the `generic-task-manager` folder.
3. Navigate to the `task-manager` directory.
4. Open `task-manager.html` in any modern web browser.

No installation or additional dependencies are required.
